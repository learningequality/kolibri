Basic Usage Guide

Installation
------------

.. warning::
  Kolibri is not yet available on PyPi. It must be `run from source <https://github.com/learningequality/kolibri>`_ for now.

You can install Kolibri as a standard package from PyPi using this command::

    $ pip install kolibri
